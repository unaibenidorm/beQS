# 🚀 Better Quick Settings (beQS) 1.1

**Better Quick Settings** is the ultimate GNOME Shell extension designed to transform your system's Quick Settings menu into a powerful, beautiful, and highly functional control center. 

Tailor-made for GNOME 48, 49, and 50, it provides a premium level of customization that feels native yet significantly more capable.

---

## 💎 Premium Features

### 🔊 Advanced Audio Control
*   **Per-App Volume Mixer**: Control individual application volumes directly from your panel.
*   **Smart Device Management**: Automatically detect, rename, and hide system audio devices (Outputs & Inputs) to keep your menu clean.
*   **Precise Filtering**: Only see the applications and hardware you care about.

### 🎵 Media & Entertainment
*   **Enhanced Media Widget**: Beautiful cover art fetching from **iTunes**, smooth scrolling titles, and dynamic layouts.
*   **CAVA Integration**: Real-time audio visualizer right in your Quick Settings.
*   **Metadata Mastery**: Intelligent track info parsing for a cleaner look.

### 📊 System Intelligence
*   **Side-Pane Notifications**: A dedicated, swipeable panel for your notifications on the left side of the menu.
*   **Weather & Environment**: Integrated weather info and a real-time **Hygrometer** for monitoring humidity.
*   **System Toggles**: Customize exactly which system buttons (VPN, WWAN, Night Light, etc.) are visible.

### 🎨 Aesthetics & UX
*   **Overlay Mode**: Experience a stunning centered menu with smooth animations.
*   **Blur & Transitions**: Native-feeling background blur and refined interaction effects.
*   **Modern Design**: Clean, minimal, and fully integrated with the latest GNOME aesthetics.

## 🛠️ Installation

### 1. Clone the Repository
```bash
git clone https://github.com/unaibenidorm/beQS.git ~/.local/share/gnome-shell/extensions/beQS@unaibenidorm
```

### 2. Compile Schemas
```bash
glib-compile-schemas ~/.local/share/gnome-shell/extensions/beQS@unaibenidorm/schemas/
```

### 3. Apply Changes
*   **X11**: Press `Alt` + `F2`, type `r`, and hit `Enter`.
*   **Wayland**: Log out and log back in.

### 4. Enable
Activate **Better Quick Settings** via the GNOME Extensions app or Extension Manager.
